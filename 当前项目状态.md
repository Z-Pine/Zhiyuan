## 🐛 已知问题

### 1. CSP错误（已解决）
**问题**: Content Security Policy阻止JavaScript执行  
**状态**: ✅ 已解决  
**解决方案**: 使用独立Chrome配置 + disable-web-security  
**文档**: `CSP问题解决方案.md`

### 2. Web平台Token存储（已解决）
**问题**: FlutterSecureStorage在Web平台不兼容  
**状态**: ✅ 已解决  
**解决方案**: Web使用SharedPreferences  
**文档**: `TOKEN_FIX_COMPLETE.md`

### 3. 推荐算法分类（已解决）
**问题**: 冲刺/稳妥/保底分类错误  
**状态**: ✅ 已解决  
**解决方案**: 修复位次比例算法

## 🔧 开发环境

### 必需软件
- Node.js v18+
- Flutter 3.x
- PostgreSQL (Neon)
- Chrome浏览器

### 启动命令

**后端**:
```powershell
cd backend
npm run dev
```

**前端**:
```powershell
cd frontend
flutter run -d chrome --web-port=8080 `
  --web-browser-flag="--disable-web-security" `
  --web-browser-flag="--user-data-dir=C:\temp\chrome-dev"
```

或使用启动脚本：
```powershell
cd frontend
.\start-dev.ps1
```
