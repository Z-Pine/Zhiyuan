### 学生管理

#### 如何添加学生

1. **进入学生管理**
   - 点击首页的"基本信息"卡片
   - 或从菜单进入学生管理

2. **点击添加学生**
   - 点击右下角蓝色"添加学生"按钮
   - 或点击右上角"+"图标

3. **填写学生信息**

   **基本信息**:
   - 姓名: 必填，学生的真实姓名
   - 性别: 必选，男生/女生

   **地区信息**:
   - 省份: 必选，默认广东
   - 城市: 选填，所在城市
   - 高中: 选填，就读高中名称

   **考试信息**:
   - 选科类型: 必选，物理类/历史类
   - 高考总分: 选填，0-750分
   - 省排名: 选填，省内位次

4. **保存学生**
   - 点击"添加学生"按钮
   - 等待保存成功提示
   - 自动返回学生列表

#### 如何选择当前学生

1. 在学生列表中
2. 点击学生卡片
3. 卡片会高亮显示
4. 右上角显示"当前"标签

#### 如何编辑学生

1. 点击学生卡片右侧的"⋮"按钮
2. 选择"编辑"
3. 修改信息
4. 点击"保存修改"

#### 如何删除学生

1. 点击学生卡片右侧的"⋮"按钮
2. 选择"删除"
3. 确认删除操作
4. 学生将被删除（功能待实现）

### 推荐 Tab

#### 如何生成推荐方案

1. **准备工作**
   - 确保已添加学生
   - 确保已选择当前学生
   - 确保学生信息完整（包括分数和位次）

2. **生成推荐**
   - 切换到"推荐"Tab
   - 查看学生信息是否正确
   - 点击"一键生成推荐方案"按钮
   - 等待2-3秒

3. **查看结果**
   - 推荐方案生成成功
   - 显示推荐概览
   - 点击"查看推荐结果"查看详情

#### 推荐结果说明

**冲刺院校**:
- 录取位次优于考生位次
- 有一定挑战性
- 录取概率: 15%-50%
- 建议: 可以冲一冲

**稳妥院校**:
- 录取位次与考生位次相当
- 录取概率较大
- 录取概率: 65%-80%
- 建议: 把握较大

**保底院校**:
- 录取位次低于考生位次
- 录取把握很大
- 录取概率: 82%-95%
- 建议: 确保录取

#### 如何查看院校详情

1. 在推荐结果页面
2. 点击院校卡片
3. 查看院校详细信息
4. 查看推荐专业
5. 查看匹配分析

## 💡 使用技巧

### 1. 完善学生信息
- 尽量填写完整的学生信息
- 分数和位次是生成推荐的关键
- 选科类型会影响推荐结果

### 2. 理解推荐结果
- 冲刺院校: 可以尝试，但有风险
- 稳妥院校: 重点关注，录取概率大
- 保底院校: 确保录取，避免滑档

### 3. 合理填报志愿
- 参考推荐结果
- 结合个人兴趣
- 考虑就业前景
- 咨询老师和家长

### 4. 使用AI追问
- 针对推荐结果提问
- 询问院校和专业信息
- 了解就业前景
- 获取填报建议

## ⚠️ 注意事项

### 1. 数据准确性
- 请确保输入的分数和位次准确
- 数据来源于历年录取数据
- 仅供参考，不作为最终决策依据

### 2. 推荐结果
- 推荐结果基于算法计算
- 实际录取受多种因素影响
- 建议结合实际情况综合考虑

### 3. 志愿填报
- 推荐方案仅供参考
- 最终填报需谨慎决策
- 建议咨询专业老师
- 关注官方招生信息

### 4. 系统限制
- 当前仅支持广东省数据
- 部分功能仍在开发中
- 建议使用Chrome浏览器
