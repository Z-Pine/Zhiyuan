## 📋 功能概述

实现了完整的学生画像问卷系统，通过5个步骤收集学生的多维度信息：

1. **性格特点** - MBTI 16型人格测评
2. **兴趣爱好** - 10个领域的兴趣选择
3. **职业倾向** - 职业发展偏好和学习风格
4. **地域偏好** - 省份和院校类型偏好
5. **家庭期望** - 家长对孩子未来的期望

## 📊 数据结构

### 提交到后端的数据格式
```json
{
  "mbti_type": "INTJ",
  "interests": ["science", "technology"],
  "abilities": [],
  "career_preferences": {
    "career_preference": "development",
    "study_style": "balanced",
    "risk_preference": "low"
  },
  "province_preferences": ["广东", "北京", "上海"],
  "university_type_preferences": ["985工程", "211工程"],
  "family_expectations": "希望孩子从事稳定的工作..."
}
```

### 数据库存储
- `mbti_type`: VARCHAR - MBTI类型代码
- `holland_code`: VARCHAR - 霍兰德代码（预留）
- `interests`: JSONB - 兴趣数组
- `abilities`: JSONB - 能力数组
- `career_preferences`: JSONB - 职业偏好对象
- `province_preferences`: JSONB - 省份偏好数组
- `university_type_preferences`: JSONB - 院校类型数组
- `family_expectations`: TEXT - 家庭期望文本
