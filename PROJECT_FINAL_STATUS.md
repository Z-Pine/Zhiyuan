## 🎉 项目概览

这是一个基于AI的高考志愿填报辅助系统，帮助广东省（及其他新高考3+1+2省份）考生科学填报志愿。

### 核心价值
- 🎯 **智能推荐**: 基于位次算法的冲刺/稳妥/保底院校推荐
- 📊 **数据驱动**: 使用真实历年录取数据（2023-2024年）
- 🎨 **个性画像**: MBTI + 霍兰德代码匹配
- 💼 **就业导向**: 专业-行业关联分析
- ⚠️ **风险评估**: 多维度录取风险评估

**算法核心逻辑**:
```javascript
位次比例 = 院校录取位次 / 考生位次

0.3-1.0  → 冲刺（院校更好，有挑战）
1.0-1.5  → 稳妥（相当，概率大）
1.5-2.5  → 保底（院校更差，把握大）
```
