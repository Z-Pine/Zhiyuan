## 📋 测试步骤

### 第1步：清除浏览器缓存
1. 打开 http://localhost:8080
2. 按 **F12** 打开开发者工具
3. 切换到 **Application** 标签
4. 点击左侧 **Storage** 下的 **Clear storage**
5. 点击 **Clear site data** 按钮
6. 刷新页面 (F5)

### 第2步：登录并查看日志
1. 在登录页面输入：
   - 手机号：**13800138000**
   - 密码：**123456**
2. 切换到 **Console** 标签（重要！）
3. 点击"登录"按钮
4. 观察Console日志，应该看到：
   ```
   ✅ 登录成功，准备保存token...
   🔑 Token已保存到SharedPreferences: eyJhbGci...
   🔐 ApiService: Token已设置到请求头: Bearer eyJhbGci...
   ✅ Token验证成功
   ```

### 第3步：使用调试工具（可选）
1. 登录成功后，在首页右上角找到 **"调试"** 按钮
2. 点击进入调试页面
3. 查看Token状态，应该显示：
   - ✅ Token已找到
   - 📱 手机号: 13800138000
   - 🔑 Token前20字符
4. 点击 **"测试获取学生列表"** 按钮
5. 应该看到成功响应（即使列表为空也是正常的）

### 第4步：添加学生
1. 返回首页
2. 点击 **"基本信息"** 卡片（蓝色图标）
3. 点击右上角 **"+"** 按钮
4. 填写学生信息：
   - 姓名：**张三**
   - 性别：**男**
   - 省份：**广东**
   - 选科类型：**物理类**
5. 点击 **"保存"** 按钮
6. 观察Console日志：
   ```
   📝 准备添加学生: 张三
   📡 POST /api/students, Authorization: Bearer eyJhbGci...
   📝 收到响应: statusCode=200
   ✅ 学生添加成功
   ```

### 第5步：验证结果
1. 应该自动返回学生列表页面
2. 列表中应该显示 **"张三"**
3. 返回首页，应该看到学生信息卡片显示 **"张三"**

## 🔍 调试技巧

### 查看localStorage中的token
在Console中执行：
```javascript
localStorage.getItem('flutter.auth_token')
```
应该返回一个长字符串（JWT token）

### 查看网络请求
1. 切换到 **Network** 标签
2. 找到 **POST /api/students** 请求
3. 点击查看详情
4. 查看 **Request Headers**
5. 确认 **Authorization** 字段格式：`Bearer <token>`

### 后端验证
如果怀疑是后端问题，可以运行测试脚本：
```bash
cd backend
node test-add-student.js
```
应该看到：
```
✅ 登录成功
✅ 添加学生成功
```
