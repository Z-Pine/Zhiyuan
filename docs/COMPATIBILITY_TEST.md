## 测试方法

### 自动化测试

```bash
# 运行所有测试
flutter test

# 运行特定设备测试
flutter test --device-id <device_id>

# 运行集成测试
flutter test integration_test/app_test.dart
```

### 手动测试

1. **安装测试**
   - 从应用商店安装
   - 从APK/IPA安装
   - 覆盖安装
   - 卸载后重新安装

2. **功能测试**
   - 按照测试用例逐一测试
   - 记录测试结果
   - 截图记录问题

3. **探索性测试**
   - 边界条件测试
   - 异常操作测试
   - 长时间使用测试
